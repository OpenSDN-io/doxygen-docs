Detailed Description

Helper class that provides default implementations of TProtocol methods.

This class provides default implementations of the non-virtual TProtocol methods. It exists primarily so TVirtualProtocol can derive from it. It prevents TVirtualProtocol methods from causing infinite recursion if the non-virtual methods are not overridden by the TVirtualProtocol subclass.

You probably don't want to use this class directly. Use TVirtualProtocol instead.
