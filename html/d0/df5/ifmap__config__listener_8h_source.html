32 //
33 // This class implements an observer for events on the IFMapTables associated
34 // with configuration items. It listens to the IFMapTables in question and
35 // puts ConfigDeltas on the change list. TableMap is a list of IFMapTable
36 // names and corresponding DBTable::ListenerIds that this class has registered.
37 //
38 // The DependencyTracker recursively evaluates dependencies as specified via a
39 // policy and pushes additional ConfigDeltas to the change list. This takes
40 // the burden of dependency tracking away from the consumers and automates it
41 // instead of being individually hand coded for each type of object.
42 //
43 // The ChangeList of ConfigDeltas is processed by the ConfigManager with
44 // which this IFMapConfigListener is associated.
45 //
