30 //
31 // This class implements an observer for events on the IFMapTables associated
32 // with configuration items. It listens to the IFMapTables in question and
33 // puts ConfigDeltas on the change list. TableMap is a list of IFMapTable
34 // names and corresponding DBTable::ListenerIds that this class has registered.
35 //
36 // The DependencyTracker recursively evaluates dependencies as specified via a
37 // policy and pushes additional ConfigDeltas to the change list. This takes
38 // the burden of dependency tracking away from the consumers and automates it
39 // instead of being individually hand coded for each type of object.
40 //
41 // The ChangeList of ConfigDeltas is processed by the ConfigManager with
42 // which this IFMapConfigListener is associated.
43 //
