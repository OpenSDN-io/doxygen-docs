9 // This class provides filtering for flow trace messages. In scaled environment
10 // tracing has following problems,
11 //
12 // 1. The trace-buffer is limited, so the messages overflow buffer very quick.
13 // Its difficult to catch traces for flows of interest
14 // 2. Logging adds latency and becomes a bottleneck
15 //
16 // Flow filtering provides mechanism allows to filter and trace only flows of
17 // interest.
18 //
19 // By default, filtering allows all flows to be traced. Tracing can be
